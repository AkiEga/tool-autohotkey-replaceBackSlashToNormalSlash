# About
"\" を "/" に置き換えるツールです。

# Usage
1. インストール方法
	1. [autohotkey](https://www.autohotkey.com)のインストール
 	2. 本リポジトリをgit clone or download
	3. autohotkeyでコンパイル(replaceBackSlashToNormalSlash.exeというexeファイルが出来る)
  
2. 使用方法
	1. replaceBackSlashToNormalSlash.exeを実行
	2. "\" の箇所を選択
 	3. "Ctrl + /" を押下
 	4. 完了("\" が "/" に置き換えられているはず)
